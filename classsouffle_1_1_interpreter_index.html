virtual void souffle::InterpreterIndex::extend ( InterpreterIndex *  )
inlinevirtual

Extend another index.

This should only affect on an EqrelIndex. Extend this index with another index, expanding this equivalence relation. The supplied relation is the old knowledge, whilst this relation only contains explicitly new knowledge. After this operation the "implicitly new tuples" are now explicitly inserted this relation.
